# 知识库目录

请将您的文献文件放在此目录下。

## 支持的格式

- **PDF**: 学术论文、书籍章节等
- **TXT**: 纯文本笔记、文档
- **Markdown** (.md): Markdown 格式的笔记和文档

## 文件组织建议

```
knowledge_base/
├── 理论基础/
│   ├── 统计学方法.pdf
│   └── 研究设计.pdf
├── 文献综述/
│   ├── 相关研究1.pdf
│   └── 相关研究2.pdf
└── 笔记/
    ├── 课程笔记.txt
    └── 阅读笔记.md
```

## 构建索引

添加新文件后，需要重新构建索引：

```bash
python build_index.py
```

索引会保存在 `outputs/` 目录中。
