# Knowledge Base Directory

Place your reference documents in this folder.

## Supported Formats

- **PDF**: Research papers, book chapters, lecture slides
- **TXT**: Plain text notes or documents
- **Markdown (.md)**: Markdown-formatted notes and documents

## Suggested Structure

knowledge_base/
├── Foundations/
│ ├── Statistical_Methods.pdf
│ └── Research_Design.pdf
├── Literature_Review/
│ ├── Related_Work_1.pdf
│ └── Related_Work_2.pdf
└── Notes/
├── Course_Notes.txt
└── Reading_Notes.md



## Build Index

After adding new files, rebuild the index:

```bash
python build_index.py
