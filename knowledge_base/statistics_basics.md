# 统计学基础知识

## 方差分析 (ANOVA)

### 什么是方差分析？

方差分析（Analysis of Variance, ANOVA）是一种统计方法，用于比较三个或更多组之间的均值差异。它通过分析数据的方差来源来确定组间差异是否显著。

### 方差分析的类型

1. **单因素方差分析**：只有一个自变量
2. **多因素方差分析**：有多个自变量
3. **重复测量方差分析**：同一受试者在不同条件下重复测量

### 重复测量方差分析 (Repeated Measures ANOVA)

重复测量方差分析（Repeated Measures ANOVA）用于分析同一受试者在不同时间点或不同条件下的测量数据。

#### 前提假设

重复测量方差分析有三个主要前提假设：

1. **正态性 (Normality)**：数据应服从正态分布
2. **独立性 (Independence)**：不同受试者的观测值应相互独立
3. **球形性 (Sphericity)**：各水平差值的方差应相等

#### 球形性假设 (Sphericity Assumption)

**球形性假设**是重复测量方差分析最重要的前提条件之一。它要求：

- 所有配对条件之间差值的方差应该相等
- 换句话说，任意两个水平之间差异的方差应该相同
- 数学表达：Var(X_i - X_j) = Var(X_k - X_l) 对于所有配对

**违反球形性假设的后果**：
- 会导致 F 统计量偏大
- 增加 I 类错误（假阳性）的概率
- 结论可能不可靠

#### Greenhouse-Geisser 校正

当数据违反球形性假设时，可以使用 **Greenhouse-Geisser 校正**：

- 这是一种保守的校正方法
- 通过调整自由度来校正 F 检验
- 校正系数 ε（epsilon）的范围：1/(k-1) ≤ ε ≤ 1
  - ε = 1 表示完全满足球形性
  - ε 越小，偏离球形性越严重
- 校正后的自由度 = 原始自由度 × ε

**使用建议**：
- 先进行 Mauchly 球形性检验
- 如果 p < 0.05，违反球形性假设
- 此时应使用 Greenhouse-Geisser 校正
- 如果 ε > 0.75，可以使用 Huynh-Feldt 校正

### t 检验 (t-test)

t 检验用于比较两组均值的差异。

#### 独立样本 t 检验
- 用于比较两个独立组的均值
- 假设：正态性、方差齐性

#### 配对样本 t 检验
- 用于比较同一组在两个时间点的均值
- 假设：差值的正态性

### 相关分析 (Correlation)

相关分析用于研究两个变量之间的线性关系。

#### Pearson 相关系数
- 范围：-1 到 +1
- r = 1：完全正相关
- r = -1：完全负相关
- r = 0：无线性相关

**解释标准**：
- |r| < 0.3：弱相关
- 0.3 ≤ |r| < 0.7：中等相关
- |r| ≥ 0.7：强相关

### 回归分析 (Regression)

回归分析用于预测一个变量（因变量）如何受到一个或多个变量（自变量）的影响。

#### 简单线性回归
- 模型：Y = β₀ + β₁X + ε
- β₀：截距
- β₁：斜率
- R²：决定系数，表示模型解释的方差比例

#### 多元线性回归
- 多个自变量预测一个因变量
- 需要检查多重共线性

## 描述统计

### 集中趋势
- **均值 (Mean)**：所有数据的平均值
- **中位数 (Median)**：数据的中间值
- **众数 (Mode)**：出现频率最高的值

### 离散程度
- **标准差 (Standard Deviation, SD)**：数据偏离均值的程度
- **方差 (Variance)**：标准差的平方
- **四分位距 (IQR)**：第 75 百分位数 - 第 25 百分位数

### 效应量 (Effect Size)

效应量衡量效应的大小，不受样本量影响。

#### Cohen's d
- 用于 t 检验
- d = (M₁ - M₂) / SD_pooled
- 解释：
  - d = 0.2：小效应
  - d = 0.5：中等效应
  - d = 0.8：大效应

## 非参数检验

当数据不满足参数检验的假设时，使用非参数检验。

### Mann-Whitney U 检验
- 独立样本 t 检验的非参数替代
- 比较两组的秩次

### Wilcoxon 符号秩检验
- 配对样本 t 检验的非参数替代
- 比较配对数据的差异

### Kruskal-Wallis 检验
- 单因素方差分析的非参数替代
- 比较三个或更多组的秩次

## 实验设计

### 组间设计 (Between-Subjects Design)
- 不同受试者分配到不同条件
- 优点：无练习效应
- 缺点：需要更多受试者

### 组内设计 (Within-Subjects Design)
- 同一受试者经历所有条件
- 优点：控制个体差异
- 缺点：可能有练习效应

### 混合设计 (Mixed Design)
- 结合组间和组内因素
- 同时考虑两种设计的优缺点
