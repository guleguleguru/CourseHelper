# 数据目录

请将您的 CSV 数据文件放在此目录下。

## 支持的格式

- **CSV**: 逗号分隔值文件

## 数据要求

1. **表头**: 文件必须包含表头行（第一行为列名）
2. **编码**: 使用 UTF-8 编码
3. **分隔符**: 使用逗号 (`,`) 作为分隔符
4. **格式**: 标准 CSV 格式

## CSV 示例

```csv
id,age,gender,bmi,group
1,25,M,22.5,control
2,30,F,24.1,treatment
3,28,M,21.8,control
4,35,F,26.3,treatment
```

## 最佳实践

### 列名建议
- 使用英文字母和数字
- 避免特殊字符和空格
- 使用下划线 `_` 分隔单词
- 示例: `age`, `bmi_value`, `treatment_group`

### 数据清洗
- 确保缺失值使用统一标记（如空字符串或 `NA`）
- 确保日期格式统一
- 确保数值列不包含非数字字符

### 文件大小
- 建议单文件不超过 100MB
- 如果数据量大，考虑拆分或使用数据库

## 使用示例

```
"展示 experiment.csv 的前 10 行"
```

## 安全说明

- 此目录下的数据仅在本地处理
- 不会上传完整数据到云端
- 仅查询内容会发送给 LLM API
